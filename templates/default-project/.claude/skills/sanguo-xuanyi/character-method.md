# 三国人物设定方法

## 一、身份设定

### 官职体系
设定角色官职时，必须符合东汉末年至三国时期的官制体系。

#### 中央官职（按品级）
| 等级 | 官职 | 职能 |
|------|------|------|
| 最高 | 丞相、大将军 | 总揽政务/最高军事长官 |
| 三公 | 太尉、司徒、司空 | 名义上的最高官员 |
| 九卿 | 太常、光禄勋、卫尉等 | 具体部门主管 |
| 将军 | 骠骑、车骑、卫将军等 | 高级军事主官 |
| 中层 | 尚书令、侍中 | 实际决策核心 |

#### 地方官职
| 官职 | 管辖范围 | 说明 |
|------|----------|------|
| 州牧/刺史 | 一州 | 最高地方长官 |
| 太守 | 一郡 | 郡级长官 |
| 县令/县长 | 一县 | 县级长官（万户以上为令） |

#### 军事职位
| 级别 | 职位示例 |
|------|----------|
| 高级 | 四征将军、四镇将军、四安将军、四平将军 |
| 中级 | 杂号将军、校尉 |
| 基层 | 都尉、军司马、曲长 |

### 身份选择建议
1. **主角**：中下层官吏更适合探案（如县令、主簿、功曹）
2. **幕后黑手**：高层权贵更有戏剧张力
3. **配角**：各阶层搭配，丰富社会面貌

## 二、势力归属

### 三大阵营
| 势力 | 核心特点 | 人物风格 |
|------|----------|----------|
| 曹魏 | 唯才是举、法度严明 | 务实、精明、重权术 |
| 蜀汉 | 匡扶汉室、仁义为本 | 忠义、理想主义 |
| 东吴 | 据江东、重世家 | 灵活、重家族利益 |

### 中立/边缘势力
- 世家大族（如颍川荀氏、琅琊诸葛）
- 隐士游侠
- 商贾、匠人
- 少数民族（乌桓、羌、山越等）

## 三、言行举止规范

### 士人风范
1. **称谓**
   - 自称：在下、某、仆、愚
   - 尊称：公、君、足下、先生
   - 官场称谓：以官职相称（府君、将军、主公）

2. **礼仪**
   - 重视冠带（士人标志）
   - 跪坐/席地而坐（胡床尚未普及）
   - 拱手、长揖、稽首等礼节

3. **言谈特点**
   - 引经据典
   - 善用比喻、典故
   - 含蓄委婉，忌直白

### 武将风格
1. **言语**
   - 较为直接
   - 重诺守信
   - 好论勇武功绩

2. **行为**
   - 重武艺、马术
   - 讲义气、重情谊
   - 快意恩仇

### 下层人物
1. **百姓**
   - 言语朴实
   - 关心温饱
   - 敬畏权贵

2. **游侠**
   - 不拘礼法
   - 重然诺
   - 行事果决

## 五、历史人物处理

### 使用原则
1. **尊重基本史实**
   - 生卒年、主要事迹不可随意更改
   - 性格可在史料基础上适度发挥
   - 虚构情节不可与已知史实矛盾

2. **留有余地**
   - 利用历史的"空白期"创作
   - 在"可能发生"的范围内虚构
   - 不写有定论的未解之谜的"答案"

3. **配角定位**
   - 历史名人适合作为背景或配角
   - 主角建议为虚构人物
   - 避免过度美化或丑化历史人物

### 历史人物参考示例
| 人物 | 可用元素 | 注意事项 |
|------|----------|----------|
| 曹操 | 多疑、诗才、用人不拘一格 | 避免写成纯粹奸臣 |
| 诸葛亮 | 谨慎、智谋、鞠躬尽瘁 | 避免过度神化 |
| 司马懿 | 隐忍、深谋、善于等待 | 注意其在不同时期的身份 |

## 六、角色关系网

### 关系类型
1. **官场关系**：上下级、同僚、政敌
2. **家族关系**：宗族、姻亲
3. **私人关系**：师徒、故交、恩仇
4. **秘密关系**：间谍、内应、私约

### 关系设计原则
1. 每个重要角色至少与3个其他角色有明确关系
2. 关系网中要有冲突点
3. 随剧情发展，关系可以变化
4. 考虑势力背景对关系的影响

## 七、角色弧光设计

### 变化模式
1. **成长型**：从无知到成熟
2. **堕落型**：从正直到黑化
3. **救赎型**：从迷失到回归
4. **坚守型**：历经考验不改初心

### 三阶段设计
```
起点状态 → 触发事件 → 内心挣扎 → 选择/行动 → 终点状态
```

### 与悬疑结合
- 角色的成长可与案件调查同步
- 真相揭示可以成为角色转变的触发点
- 角色的选择影响案件走向
