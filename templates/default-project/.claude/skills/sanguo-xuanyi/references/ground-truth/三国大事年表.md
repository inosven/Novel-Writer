## 创作时间线检查要点

### 常见时间错误

1. **关羽之死时间**
   - ❌ 错误：建安二十三年关羽北伐
   - ✓ 正确：建安二十四年（219年）

2. **赤壁之战时间**
   - ❌ 错误：建安十二年赤壁之战
   - ✓ 正确：建安十三年（208年）

3. **三国建立时间**
   - 魏：220年曹丕称帝
   - 蜀：221年刘备称帝
   - 吴：229年孙权称帝（222年称吴王）

4. **诸葛亮出山时间**
   - ❌ 错误：赤壁之战后诸葛亮出山
   - ✓ 正确：建安十二年（207年）三顾茅庐

### 人物存活检查

写作时需确认该时间点人物是否存活：

```
示例：建安十三年（208年）赤壁之战时
✓ 存活：曹操、刘备、孙权、诸葛亮、周瑜、关羽、张飞
✗ 已亡：孙坚（191亡）、孙策（200亡）、袁绍（202亡）
✗ 未出生：无关键人物
```
