# 审稿规则

## 一、审核等级

### 错误分级
| 等级 | 说明 | 处理方式 |
|------|------|----------|
| **致命错误** | 违背核心史实、严重逻辑漏洞 | 必须修改 |
| **严重错误** | 明显的历史/人物错误 | 强烈建议修改 |
| **一般问题** | 文风不统一、节奏问题 | 建议修改 |
| **轻微问题** | 用词可优化、描写可增强 | 可选修改 |

## 二、历史准确性审核

### Ground Truth 校验项

#### 1. 官职体系
- [ ] 官职名称是否存在于东汉末年
- [ ] 官职品级是否正确
- [ ] 任命权限是否合理（如：只有天子/丞相能任命某些官职）

#### 2. 时间线
- [ ] 事件年份是否正确
- [ ] 人物在该时间是否存活
- [ ] 人物在该时间的身份/官职是否正确
- [ ] 季节/月份与事件是否匹配

#### 3. 地理
- [ ] 地名是否为当时称呼
- [ ] 州郡隶属关系是否正确
- [ ] 距离/行程时间是否合理
- [ ] 地理特征描写是否准确

#### 4. 人物事实
- [ ] 历史人物的基本信息是否正确（字、籍贯、主要经历）
- [ ] 人物关系是否符合史实
- [ ] 人物的性格表现是否与史料记载相符

### 常见历史错误检查表
| 检查项 | 正确 | 常见错误 |
|--------|------|----------|
| 曹操称丞相时间 | 建安十三年(208) | 写成更早 |
| 刘备称帝时间 | 章武元年(221) | 与其他事件时间冲突 |
| 赤壁之战年份 | 建安十三年(208) | 年份错误 |
| 三国正式成立 | 220-222年间 | 过早使用"三国" |
| 关羽败亡时间 | 建安二十四年(219) | 时间线混乱 |

## 三、悬疑逻辑审核

### 线索审核
- [ ] 所有最终用于推理的线索是否已向读者展示
- [ ] 线索展示是否过于明显或过于隐蔽
- [ ] 是否存在无关紧要却占篇幅的假线索
- [ ] 线索与结论的逻辑链是否完整

### 推理审核
- [ ] 推理过程是否合乎逻辑
- [ ] 是否存在跳跃性推理
- [ ] 结论是否能从已知信息推出
- [ ] 是否存在更合理的替代解释未被排除

### 凶手/真相审核
- [ ] 真凶是否有作案动机
- [ ] 真凶是否有作案能力
- [ ] 真凶是否有作案时间
- [ ] 作案手法是否可行

### 伏笔审核
- [ ] 所有伏笔是否都有回收
- [ ] 回收时机是否恰当
- [ ] 是否存在未回收的悬念

## 四、人物一致性审核

### 知识图谱校验
- [ ] 人物姓名、字、籍贯是否前后一致
- [ ] 人物年龄在时间线中是否正确变化
- [ ] 人物关系描述是否前后一致
- [ ] 人物官职变化是否合理

### 性格一致性
- [ ] 人物行为是否符合已建立的性格
- [ ] 若有反常行为，是否有合理解释
- [ ] 人物说话方式是否一致
- [ ] 人物的决策逻辑是否符合其人设

### 角色弧光审核
- [ ] 角色转变是否有充分铺垫
- [ ] 转变触发点是否明确
- [ ] 转变过程是否自然
- [ ] 最终状态是否合理

## 五、文风审核

### 古风规范检查
- [ ] 是否使用了现代禁用词
- [ ] 对话是否符合人物身份
- [ ] 描写用词是否准确
- [ ] 句式是否符合古风

### 一致性检查
- [ ] 全文文风是否统一
- [ ] 不同场景的语言风格是否恰当
- [ ] 叙述视角是否一致

### 具体检查项
```
现代词汇检测：
- 搜索关键词：老婆、老公、OK、厉害、牛、
  尴尬、纠结、吐槽、秒杀、网红...

文风检测：
- 过长句子（超过30字）
- 现代句式
- 网络流行语
```

## 六、节奏与结构审核

### 章节审核
- [ ] 章节长度是否均衡
- [ ] 每章是否有明确的推进
- [ ] 章首钩子是否有效
- [ ] 章末悬念是否恰当

### 整体节奏
- [ ] 紧张与舒缓是否交替
- [ ] 高潮位置是否合适
- [ ] 信息揭示节奏是否得当
- [ ] 是否存在拖沓或仓促
