# 古风对话范例

## 一、日常对话

### 问候寒暄

**正式场合**
```
甲："久仰大名，今日得见，幸何如之！"
乙："不敢，不敢。在下久闻先生高才，今日一见，果然名不虚传。"
```

**熟人相见**
```
甲："元直兄，别来无恙？"
乙："托兄台的福，一切安好。兄台近来可好？"
甲："马马虎虎，勉强度日罢了。"
```

**偶遇**
```
甲："咦？这不是王兄么？真是巧了。"
乙："哈哈，何止是巧！正想着找你商议一事，你倒自己送上门来了。"
```

### 请求帮助

**对上级**
```
"下官有一事相求，还望大人成全。"
"末将斗胆，恳请将军恩准。"
```

**对平辈**
```
"有一事想请兄台帮忙，不知方便否？"
"此事若无你相助，恐难成事。"
```

**对下级**
```
"有件事交给你去办。"
"这差事就劳烦你跑一趟了。"
```

### 道谢道歉

**道谢**
```
"多谢，多谢！此恩此德，没齿难忘。"
"承蒙相助，感激不尽。"
"区区小事，何足挂齿。"（回应）
```

**道歉**
```
"是在下唐突了，还望见谅。"
"此事是我欠考虑，向兄台赔罪。"
"小事一桩，何必放在心上。"（回应）
```

## 二、官场对话

### 上下级对话

**下级禀报**
```
"禀大人，城外发现可疑人等。"
"启禀将军，粮草已筹备完毕，随时可以启程。"
"回丞相，此事已查明，详情容臣细禀。"
```

**上级询问**
```
"此事你怎么看？"
"依你之见，当如何处置？"
"你可有把握？"
```

**上级吩咐**
```
"此事就交给你去办，不可有误。"
"限你三日之内查明此案，否则军法从事。"
"好生去做，做好了自有你的好处。"
```

**下级领命**
```
"诺！"
"下官遵命。"
"末将领命。"
"卑职这就去办。"
```

### 同僚对话

**征求意见**
```
"仲达以为此计可行否？"
"公达兄高见，在下愿闻其详。"
```

**表达异议**
```
"此言差矣。依我之见……"
"恕我直言，这恐怕不妥。"
"此计虽妙，然有一虑……"
```

**赞同**
```
"公言甚是。"
"正合我意。"
"妙！妙！此计大妙！"
```

## 三、谋略对话

### 献策

```
甲："主公，臣有一计，或可退敌。"
乙："哦？愿闻其详。"
甲："敌军远来，利在速战。我军若坚壁清野，不与其战，待其粮尽自退。此时再以精兵袭其后路，必获全胜。"
乙："若敌军不退，强攻我城，又当如何？"
甲："主公勿忧。敌众我寡，攻城非其所长。况我城池坚固，粮草充足，守上一月不成问题。敌军无法速胜，军心必乱，届时便是我军反击之机。"
```

### 辩论

```
甲："战！当此之时，敌骄我奋，正是一战之机！"
乙："不可！敌军势大，贸然出战，无异于以卵击石。"
甲："兵法云：'出其不意，攻其不备。'敌军虽强，然轻敌已久。我军若趁夜偷袭，必可建功。"
乙："话虽如此，然我军兵少将寡，一旦失利，后果不堪设想。依我之见，不若固守待援。"
丙："二位所言皆有道理。依我之见，不若……"
```

### 试探

```
甲："听闻阁下与刘皇叔有旧？"
乙："不过是同乡之谊，算不得什么。"
甲："哦？那阁下对当今天下大势，有何高见？"
乙："天下大势，分久必合，合久必分。在下不过一介布衣，哪敢妄议国事。"
甲："阁下过谦了。以阁下之才，若肯出仕，必可大展宏图。"
乙："多谢抬爱。只是在下淡泊名利，无意功名。"
```

## 四、探案对话

### 询问证人

```
甲："案发当夜，你在何处？"
乙："回大人，小人当夜在家中安睡。"
甲："可有人证？"
乙："小人独居，并无人证。"
甲："那你如何证明自己所言非虚？"
乙："这……小人……"
甲："实话实说，否则以伪证论处！"
```

### 逼问嫌犯

```
甲："张三，你还不招么？"
乙："大人明鉴，小人冤枉啊！"
甲："冤枉？这匕首是从你家搜出的，死者身上的伤口与匕首吻合。证据确凿，你还敢狡辩？"
乙："这……这是有人陷害小人！"
甲："陷害？谁会陷害你？你与死者素有旧怨，前日还当众威胁要取他性命。这些，你作何解释？"
```

### 推理对话

```
甲："依你之见，凶手是谁？"
乙："凶手……恐怕就在堂上诸人之中。"
甲："何以见得？"
乙："其一，凶手深知死者行踪，定是熟人。其二，凶手能避开巡夜军士，说明熟悉府中布局。其三，那凶器乃府中所有，外人无从得之。综合以上三点，凶手必是府中之人无疑。"
```

## 五、情感对话

### 惜别

```
甲："此去山高水远，不知何日再见。"
乙："兄台保重。待到功成之日，你我再把酒言欢。"
甲："借你吉言。也望兄台珍重，切勿以身犯险。"
乙："放心，我自有分寸。"
```

### 愤怒

```
"岂有此理！此贼竟敢如此！"
"可恶！此仇不报，我誓不为人！"
"混账！你好大的胆子！"
```

### 悲伤

```
"天不佑我，何以至此……"
"早知今日，悔不当初啊……"
"他……他真的去了？"
```

### 惊讶

```
"什么？！此话当真？"
"这……这怎么可能？"
"竟有此事？"
```

## 六、对话禁忌

### 避免使用

| 现代表达 | 古风替代 |
|----------|----------|
| 好的/OK | 诺/善/好 |
| 没问题 | 此事包在我身上 |
| 你确定？| 此话当真？/你可有把握？ |
| 怎么了？| 何事？/出了何事？ |
| 算了 | 罢了 |
| 随便 | 随意/悉听尊便 |
| 厉害 | 了得/高明/端的厉害 |
| 尴尬 | 面有难色/窘迫 |

### 身份匹配

| 身份 | 自称 | 对人称呼 |
|------|------|----------|
| 皇帝 | 朕 | 卿/爱卿 |
| 大臣对皇帝 | 臣/微臣 | 陛下/圣上 |
| 丞相 | 某/孤（曹操特例）| 君/公 |
| 将军 | 某/末将 | 将军/大人 |
| 文官 | 下官/卑职 | 大人/府君 |
| 平民 | 小人/草民 | 大人/老爷 |
| 女子 | 妾/奴家 | 郎君/公子 |

## 七、对话节奏

### 紧张时（短促）
```
"来人！"
"何事？"
"有刺客！"
"在哪？"
"东墙！"
"追！"
```

### 从容时（舒缓）
```
"今日天气甚好，不若去郊外走走？"
"也好。听闻城南新开了一处茶坊，倒是可以去看看。"
"哦？那便去看看吧。"
```

### 辩论时（层层递进）
```
"此计不可。"
"为何不可？"
"风险太大。"
"若不冒险，如何取胜？"
"可若失败……"
"不入虎穴，焉得虎子？"
```
