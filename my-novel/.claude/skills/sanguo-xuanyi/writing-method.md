# 写作技法指南

## 一、叙事视角

### 推荐视角：第三人称有限视角
- 跟随主角视角，读者与主角获取信息同步
- 便于制造悬念和意外
- 可在关键时刻切换到其他角色视角（但需谨慎）

### 视角切换原则
1. **章节边界切换**：不在章节中途随意切换
2. **必要性原则**：只在必须展示主角视角外信息时切换
3. **公平原则**：不利用切换刻意隐瞒关键信息

## 二、场景写作

### 场景类型与技法

#### 1. 朝堂场景
- **氛围营造**：庄严、压抑、暗流涌动
- **对话技巧**：
  - 言辞考究，暗藏机锋
  - 善用沉默、眼神等无声交流
  - 层层推进，步步紧逼
- **空间描写**：殿堂布局、官员序列、礼仪动作

#### 2. 探案场景
- **现场描写**：
  - 由远及近，由整体到细节
  - 通过主角视角引导读者观察
  - 重要线索自然融入描写
- **推理展示**：
  - 内心独白展示思考过程
  - 与助手对话阐释推理
  - 避免一次性长篇推理（分散到多个场景）

#### 3. 对峙场景
- **张力构建**：
  - 前期铺垫双方立场
  - 节奏由慢到快
  - 善用环境描写映衬气氛
- **高潮处理**：
  - 真相揭示要有冲击力
  - 留白让读者回味
  - 收尾干净利落

#### 4. 日常场景
- **功能**：
  - 缓解紧张节奏
  - 展示人物关系
  - 暗埋伏笔和线索
- **技巧**：
  - 不可纯粹闲笔
  - 细节中藏信息
  - 体现时代生活气息

### 战争场景（如涉及）
- 宏观与微观结合
- 战略层面与个体体验交织
- 不过度美化或血腥化
- 符合历史上的战争规模和方式

## 三、悬念技法

### 钩子类型

#### 1. 章首钩子
- **悬念延续型**：延续上一章末尾的紧张
- **场景切入型**：以紧张或神秘场景开篇
- **问题提出型**：抛出疑问引发好奇

#### 2. 章末钩子
- **危机悬停型**：主角陷入危机，戛然而止
- **发现揭示型**：重大发现刚露端倪
- **预示型**：暗示即将发生的事件

### 信息控制

#### 知情差设计
| 类型 | 读者 | 角色 | 效果 |
|------|------|------|------|
| 悬念 | 不知 | 不知 | 共同探索的紧张感 |
| 戏剧反讽 | 知道 | 不知 | 为角色担忧/期待 |
| 惊奇 | 不知 | 知道 | 揭示时的震撼 |

#### 揭示节奏
- 小线索：每1-2章
- 中型发现：每3-5章
- 重大转折：每幕一次
- 真相揭示：结局

## 四、对话写作

### 基本原则
1. **个性化**：每个角色有独特说话方式
2. **目的性**：每句对话都有其作用
3. **潜台词**：表面意思与实际含义的差异
4. **推动性**：对话推动情节或揭示信息

### 对话标签
```
# 常用对话标签
道、言、曰、说、问、答、笑道、叹道、冷哼

# 动作配合
拱手道、起身道、沉声道、附耳道

# 避免
频繁使用现代口语标签
```

### 对话密度
- 探案/对峙场景：对话为主（60-70%）
- 叙事/描写场景：对话为辅（30-40%）
- 总体比例控制在30-40%

## 五、节奏控制

### 松紧交替
```
紧张（案件发生）
  ↓
稍缓（调查准备）
  ↓
紧张（发现线索/遇阻）
  ↓
舒缓（人物互动/日常）
  ↓
高潮（重大发现/对峙）
  ↓
余波（消化整理）
```

### 句式配合
- **紧张场景**：短句为主，节奏加快
- **舒缓场景**：长短结合，从容叙述
- **高潮场景**：短句冲击，适当留白

### 章节长度
- 标准章节：4000-6000字
- 高潮章节：可适当延长
- 过渡章节：可适当缩短
- 保持整体平衡

## 六、伏笔技术

### 埋设方法
1. **背景融入**：藏在环境描写中
2. **对话暗示**：角色无意说出的话
3. **行为细节**：看似普通的举动
4. **道具伏笔**：特定物品的出现

### 回收时机
- 小伏笔：3-5章后回收
- 中伏笔：一幕内回收
- 大伏笔：跨幕回收
- 主线伏笔：结局回收

### 检查清单
- [ ] 每个伏笔都有回收计划
- [ ] 伏笔不过于明显
- [ ] 回收时有"原来如此"的感觉
- [ ] 没有遗漏未回收的伏笔

## 七、修改与润色

### 第一遍：结构检查
- 线索布局是否合理
- 节奏是否均衡
- 伏笔是否完整

### 第二遍：细节检查
- 历史细节是否准确
- 人物言行是否一致
- 逻辑是否自洽

### 第三遍：文字润色
- 语言是否符合古风
- 是否有累赘表达
- 对话是否自然生动
