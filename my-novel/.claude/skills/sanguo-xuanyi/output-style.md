# 古风文笔规范

## 一、用词规范

### 禁用词汇
以下词汇为现代用语，在三国背景下不可使用：

| 类别 | 禁用词 | 替代词 |
|------|--------|--------|
| 称谓 | 老婆、老公 | 夫人、娘子、夫君、郎君 |
| 称谓 | 爸爸、妈妈 | 父亲、母亲、阿父、阿母 |
| 动作 | OK、好的 | 诺、善、如此甚好 |
| 物品 | 椅子（胡床普及前）| 席、榻、胡床（限特定场景）|
| 食物 | 馒头（宋代才有）| 蒸饼、胡饼 |
| 量词 | 一下子、一会儿 | 须臾、片刻、顷刻 |
| 连词 | 然后、接着 | 继而、遂、乃 |
| 程度 | 非常、特别 | 甚、极、颇 |

## 二、句式规范

### 推荐句式
1. **倒装句**
   - "此事，某早有察觉"（宾语前置）
   - "何以见得？"（疑问句倒装）

2. **省略句**
   - "（某）愿闻其详"
   - "（此事）当从长计议"

3. **判断句**
   - "此乃曹公之计也"
   - "诚如君所言"

### 避免句式
1. **现代口语句**
   - ❌ "我觉得这事儿不太对"
   - ✓ "此事颇为蹊跷"

2. **过长复杂句**
   - 每句控制在20字以内为宜
   - 复杂逻辑用多个短句表达

3. **网络用语**
   - 绝对禁止任何网络流行语

## 三、对话规范

### 对话格式
```
标准格式：
"对话内容，"他道。
"对话内容。"

带动作：
他拱手道："承蒙府君厚爱。"
他站起身来，沉声道："此事休要再提。"
```

### 对话风格对照

#### 文官/谋士
```
特点：言辞文雅，善用典故
示例：
"明公且听某一言。昔者管仲相齐，法令必行，故能九合诸侯。今若欲图大事，当先正其本。"
```

#### 武将
```
特点：直接豪迈，重义轻文
示例：
"大丈夫行事，何须瞻前顾后！此贼既已落网，杀了便是！"
```

#### 谋算时
```
特点：含蓄深沉，话中有话
示例：
"此人来意，恐非表面这般简单。"他微微眯眼，"府君可曾留意，他言及粮草时，目光往何处去了？"
```

#### 审讯时
```
特点：步步紧逼，逻辑严密
示例：
"你说案发时在家中安睡，然邻人却言，亥时见你往东而去。"他将供词掷于案上，"东面，恰是死者居所。你作何解释？"
```

## 四、描写规范

### 环境描写

#### 宫殿/官衙
```
示例：
正殿巍峨，飞檐如翼。两列甲士肃立，持戟而候。殿中烛火通明，将那雕梁画栋映得金碧辉煌。
```

#### 市井
```
示例：
街市喧阗，行人往来如织。路旁酒肆茶坊鳞次栉比，吆喝叫卖之声不绝于耳。
```

#### 战场
```
示例：
金鼓齐鸣，杀声震天。但见旌旗蔽日，矛戟如林，两军将士已战作一团。
```

### 人物描写

#### 初登场
```
示例：
那人年约三旬，身长八尺，剑眉星目，颔下三缕长髯。一袭青衫虽已洗得发白，却难掩通身儒雅气度。
```

#### 动态描写
```
示例：
他猛然站起，袍袖一振，厉声道："住口！"那声势，竟将满座宾客皆震得噤若寒蝉。
```

### 心理描写

#### 内心独白
```
示例：
他心下暗忖：此人话中有话，分明是在试探。且按兵不动，看他还有什么后招。
```

#### 情绪表达
```
示例：
一股怒意自心底升起，他强自按捺，面上却仍是那副波澜不惊的模样。
```

## 五、常见错误

### 历史硬伤
| 错误 | 说明 | 正确写法 |
|------|------|----------|
| 喝茶（饮茶方式） | 三国时饮茶不加泡法 | 煮茶、茶汤 |
| 坐椅子 | 汉末以席地跪坐为主 | 坐于席上、跪坐 |
| 银子交易 | 汉末以铜钱、布帛为主 | 钱、帛、五铢钱 |
| 称臣妾 | 臣妾是清朝用语 | 妾、贱妾 |
| 吃面条（现代做法）| 当时面食多为汤饼 | 汤饼、索饼 |

### 文风不统一
- 避免在古风叙述中突然出现现代表达
- 保持全文语言风格一致
- 人物对话可略有差异，但不可过于现代

### 过度文言
- 不必每句都是文言文
- 以"半文半白"为宜
- 确保读者能够理解

## 六、参考标准

### 文风参考等级
1. **典雅**：接近史书笔法，用于严肃场景
2. **雅正**：文白相间，主要使用风格
3. **通俗**：偏白话，用于下层人物对话

- [ ] 无现代词汇误用

- [ ] 对话符合人物身份
- [ ] 描写用词准确
- [ ] 整体风格统一
